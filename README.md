## Servicios
- Prescripción comparendos (Art. 159 Ley 769/2002)
- Prescripción predial (Art. 817 E.T.)
- Prescripción deudas (Art. 2536 Código Civil)
- Eliminación reportes Datacrédito (Ley 1266/2008)

## Stack

- Deploy: Vercel o GitHub Pages
- Sin backend, sin framework
